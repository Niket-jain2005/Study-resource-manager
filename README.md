# 📚 Study Resource Manager

This is a command-line based project I built using Python to manage study resources like books.

The idea behind this project was simple — I wanted to create something practical while learning Python and data handling. Instead of just practicing small programs, I tried to build a complete system where I can add, search, update, and manage books easily.

---

## 🚀 What this project can do

* Add new books with details
* View all saved books
* Search books using keywords
* Filter books by price range
* Sort books (low to high / high to low)
* Update existing book details
* Delete books
* View basic statistics like:

  * total books
  * average price
  * highest & lowest priced book

---
